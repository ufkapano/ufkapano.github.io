PLIKI KONFIGURACYJNE

Zakładam, że lokalne repo w katalogu myproject jest skojarzone ze zdalnym repo ufkapano/myproject w serwisie Bitbucket. 

$ cat ~/.gitconfig                      # ustawienia użytkownika
[user]
	name = Andrzej Kapanowski
	email = andrzej.kapanowski@uj.edu.pl

$ cat myproject/.git/config             # ustawienia dla repo
[core]
	repositoryformatversion = 0
	filemode = true
	bare = false
	logallrefupdates = true
[remote "origin"]
	url = https://ufkapano@bitbucket.org/ufkapano/myproject.git
	fetch = +refs/heads/*:refs/remotes/origin/*
[branch "master"]
	remote = origin
	merge = refs/heads/master
