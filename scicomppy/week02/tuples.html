Tuples

INTRO

Tuples are immutable heterogeneous sequences. 

+----------------------+------------------------------+
| Operation            | Meaning                      |
+----------------------+------------------------------+
| T = () ; T = tuple() | empty tuple                  |
| T = (0,)             | one item                     |
| T = tuple([0])       | one item                     |
| T = (1, 3, 5, 7)     | four items                   |
| T = tuple(iterable)  | make tuple from iterable     |
| len(T)               | length                       |
| T1 + T2              | concatenation                |
| T * n, n * T         | repetition                   |
| T[i]                 | index                        |
| T[i:j]               | slice (new tuple)            |
| T[i:j:k]             | slice (new tuple)            |
| for item in T: pass  | iteration                    |
| item in T            | containing                   |
| item not in T        | containing                   |
| T.index(item)        | first occurrence             |
| T.count(item)        | number of occurrences        |
| max(T), min(T)       | largest|smallest item of T   |
| del T                | remove the variable T        |
| (x, y) = (a, b)      | x = a ; y = b   # assignment |
+----------------------+------------------------------+
