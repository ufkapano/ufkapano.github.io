INTRODUCTION

A 'match' statement takes an expression and compares its value to successive patterns given as one or more case blocks. This statement was introduced in Python 3.10. 

# The simplest form of a 'match' statement.

def http_error(status, flag=False):
    match status:   # header line
        case 400:
            return "Bad request"
        case 401 | 403 | 404:   # several literals can be combined using |
            return "Not allowed"
        case 404:
            return "Not found"
        case 418 if flag:   # using a guard
            return "I'm a teapot"
        case _:   # optional, never fails to match
            return "Something's wrong with the internet"
